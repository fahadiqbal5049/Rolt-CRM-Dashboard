## Getting Started

1. *Install dependencies:*
   
   npm install
   

2. *Start the development server:*
   
   npm run dev
   

3. *Build for production:*
   
   npm run build
   

4. *Preview the production build:*
   
   npm run preview
   

## Linting

This project uses ESLint with TypeScript and React plugins. To run lint checks:


npm run lint
